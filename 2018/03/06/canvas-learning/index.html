globalCompositeOperation

探照灯小demo主要借助canavs上的globalCompositeOperation属性完成，该属性设定了在画新图形时采取的策略，一共有12种方式，可见mdn文档介绍。需要注意的是源图形和目标图形组合时的色彩和透明度的计算和我们想象中可能有点不太一样，此文档中列举了计算的通用公式。

αo = αs * Fa + αb * Fb
co = αs * Fa * Cs + αb * Fb * Cb

co是组合操作之后的最终颜色，且包含alpha值。αs是源图形的透明度值，Fa由组合类型决定，Cs是源图形的颜色值，不包含颜色透明度。目标图形参数说明同上。

下表是从该文档中整理出来的12中操作类型的最终参数

在demo(23行和127行）中参数如下

αs = 0.9
αb = 1
Fa = 0
Fb = 0.1
αo = 0.9 * 0 + 1 * 0.1 = 0.1

蒙层透明度是0.9，中间组合部分透明度是0.1，因此还能看出镂空效果。

但是如果想要底下的图片能看的更清晰而将蒙层透明度降低那效果会完全不一样。比如我将蒙层透明度降低到0.5，参数计算如下。

αs = 0.5
αb = 1
Fa = 0
Fb = 0.5
αo = 0.5 * 0 + 1 * 0.5 = 0.5

此时蒙层透明度是0.5，中间组合部分计算出来的透明度也是0.5，在颜色一致的情况下，看不出任何效果，仿佛是globalCompositeOperation属性没起作用（不明所以时的我一阵恐慌，以为是自己使用姿势不对😂）。

如果我们继续降低蒙层的透明度到0.2，参数计算如下

αs = 0.2
αb = 1
Fa = 0
Fb = 0.8
αo = 0.2 * 0 + 1 * 0.8 = 0.8

效果将如下图所示，中间组合部分的颜色反而比周围的都要深，至少刚开始不知道上述的计算方式时，这种效果是与我对xor的认知是不符的（就是自以为是的以为不管怎么样应该都是出现中间镂空效果😂）
